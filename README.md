# Headbang.app

![Screenshot](screenshot.png)

## Running

1. git clone git@github.com:knoopx/headbang.app.git headbang-app
2. cd headbang-app
3. bundle
4. rake dist
3. open dist/osx/Headbang.app
4. point to your headbang server

## Development

1. git clone git@github.com:knoopx/headbang.app.git headbang-app
